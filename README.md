# Shopping List with Immer

This React project demonstrates complex state management using the `useImmer` hook.

## Features

- Add shopping list items
- Update item quantity and nested notes
- Remove items from the list
- Manage nested state using Immer draft updates

## Technologies Used

- React
- Vite
- Immer
- use-immer
- CSS

## How to Run

```bash
npm install
npm run dev
